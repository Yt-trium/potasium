potasium
========

befunge interpreter and debugger


Debug Command
=============
- SPACE   > STEP
- ENTER   > RUN
- Z       > CURSOR UP
- S       > CURSOR DOWN
- q       > CURSOR LEFT
- d       > CURSOR RIGHT
- TAB     > CREAT BREAKPOINT
