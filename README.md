Potasium
========

Befunge interpreter and debugger


Debug Command
=============
- SPACE   > STEP
- ENTER   > RUN
- RETURN  > GOTO
- Z or W  > CURSOR UP
- S       > CURSOR DOWN
- Q or A  > CURSOR LEFT
- D       > CURSOR RIGHT
- TAB     > CREAT/REMOVE BREAKPOINT
- E		  > EDIT CHAR
